# /compact-work — コンテキスト圧縮モードで作業する

引数: $ARGUMENTS（対象タスクの説明 or 空＝現在の作業に適用）

## 実行内容

context-compression スキルに従い、以下を徹底して作業する。

1. **出力の3層化**: 結論（1-3行）→ 根拠（5-10行、ファイル名:行番号つき）→ 詳細はファイル書き出し。会話に30行超の生データを展開しない
2. **探索の禁止**: ファイルを読む前に grep/glob で対象を絞る。「全体を読んでから」を行わない
3. **スクリプト化**: 集計・検索・変換など決定論的な作業は使い捨てスクリプトにして実行し、トークンを推論に温存する

引数があればそのタスクを上記規約で実行する。空なら現在進行中の作業に規約を適用し、直前の冗長な出力があれば3層形式に圧縮し直す。
