# /token-check — トークン使用量の確認と最適化提案

引数: なし

## 実行内容

1. セッションのトークン使用量を表示（Claude Code では `/cost`。他エージェントでは同等のコスト確認手段を使う）

2. 使用量が多い場合、原因を分析して提案:
   - 読み込んだファイルの一覧と行数
   - 「全体読み込み」が発生していた箇所
   - 次のタスクで分割すべき単位の提案

3. **CLAUDE.mdの最適化提案**（現プロジェクトの構成を見て）
   - 不要なディレクトリのignore追加
   - スコープ外ファイルの明示

4. **/clear のタイミング提案**
   - 今セッションでコンテキストをリセットすべき論理的区切りを示す

5. **MCPの見直し確認**
   - 現在有効なMCPを列挙（1MCP ≈ 数千トークン）
   - 不要なものがあれば無効化を提案
