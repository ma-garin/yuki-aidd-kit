# /plan — 方針の確定（実装モード解除・探索許可）

引数: $ARGUMENTS（プロジェクト概要 or 調査対象）

## 実行内容

調査と方針確定のフェーズ。実装モードを解除して探索（Read/Grep/Glob）を許可し、方針を固める。

1. **実装モードを解除**
   - `.claude/mode` が存在すれば削除する（`rm -f .claude/mode`）。これで `block-explore.sh` hook が探索を再び許可する
2. **調査 → 方針提示**
   - OPERATING-MODE.md「実装ループ」に従い、対象ファイルを grep/glob で絞ってから読む
   - 判断軸＋選択肢比較の形式で方針を提示し、合意を取る
3. **PLAN.md を生成/更新**
   - 確定した方針・対象ファイル・検証手順を記載する（次フェーズの実装がこれだけを見て進められる粒度）
4. 方針が固まったら `/implement` で実装モードに入る

**`.claude/mode` は存在すれば必ず削除すること**（探索許可状態へ戻す）。
